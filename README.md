# Noxilith 🌑

**Noxilith** — личная база заметок и мыслей в духе Obsidian: связи между заметками, граф знаний, календарь задач и таймер фокуса. Работает полностью в браузере — все данные хранятся локально (localStorage), а при желании синхронизируются через облако (Supabase).

**🌐 Приложение:** https://noxilith.vercel.app

## Возможности

- 📝 **Заметки с Markdown** — редактор + предпросмотр, `[[вики-ссылки]]` с автодополнением, `#теги`, обратные ссылки (backlinks)
- 🕸 **Граф знаний** — интерактивная визуализация связей (d3-force) в 5 стилях: классика, осколки кристалла, неон, созвездие, соты
- 🎨 **6 цветовых тем** (правило 60/30/10) + светлый/тёмный режим
- 📅 **Календарь** — задачи по датам + отметки дней создания заметок
- ⏱ **Таймер фокуса** — помодоро 25/5/15 с плавающим виджетом
- 🗂 **Архив** — выполненные задачи и корзина с восстановлением
- 📊 **Статистика** — heatmap активности в стиле GitHub, топ тегов
- ⌘K **Командная палитра** — быстрый поиск и действия
- 🔄 **Экспорт**: JSON (полный бэкап), Markdown, TXT, Excel, PowerPoint, Word, PDF
- 📥 **Импорт** (drag-and-drop): JSON, .md, .txt, .docx, .pdf, .xlsx, .csv
- ☁️ **Облачная синхронизация** (опционально) — вход по email/паролю, заметки и задачи синхронизируются между устройствами (Supabase + RLS)

## Стек

- **Vite + React 19 + TypeScript**
- **Tailwind CSS 4 + shadcn/ui**
- d3-force (граф), marked + DOMPurify (markdown), cmdk (палитра)
- xlsx, pptxgenjs, docx, jspdf, jszip — экспорт; mammoth, turndown, pdfjs-dist — импорт
- Supabase — опциональная облачная синхронизация (auth + Postgres с RLS)

## Запуск локально

```bash
bun install        # или npm install
bun run dev        # дев-сервер на http://localhost:5173
bun run build      # продакшен-сборка в dist/
```

> Шаблон содержит интеграцию с Convex (`convex/`). Для чисто локального запуска она не нужна — приложение работает без бэкенда. При добавлении базы данных см. `.env.example`.

## Данные

Все данные живут в `localStorage` браузера:

- `noxilith.notes.v1` — заметки
- `noxilith.tasks.v1` — задачи
- `noxilith.prefs.v1` — тема и палитра

Полный бэкап: ⚙️ Экспорт → JSON. Восстановление: Импорт → JSON.

### Облачная синхронизация

Вход не обязателен — без аккаунта всё работает локально. После входа (кнопка ☁️ в левой панели):

- при входе локальные и облачные данные сливаются (новее — побеждает);
- дальше каждое изменение автоматически уходит в облако;
- на другом устройстве достаточно войти под тем же аккаунтом.

Бэкенд — [Supabase](https://supabase.com): таблицы `notes` и `tasks` с Row Level Security (каждый пользователь видит только свои строки). Для своего инстанса задай `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY` (см. `src/lib/supabase.ts`, схема — в `supabase/migrations`).

## Деплой
